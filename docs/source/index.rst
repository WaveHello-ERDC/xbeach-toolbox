XBeach Python toolbox
==========================================



With the XBeach toolbox a XBeach model can be setup with Python and the ouput can be analysed. On this page the various Python functions are descibed. 

The different functionalities are described with a few examples. First an 1D model setup is created with the toolbox. The next pages show how a 2D model setup can be created. Lastly, the postprocessing is demonstrated. All the functions are descibed at the Functions page.
